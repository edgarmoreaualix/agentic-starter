# orch-04-webapp frontend

This static app fetches:
- `/backend/apps/orch-04-webapp/api.json`
- `/frontend/apps/orch-04-webapp/specialist.txt`

## Run locally
From repo root, run any static file server, for example:

```bash
python3 -m http.server 8080
```

Then open:
- `http://localhost:8080/frontend/apps/orch-04-webapp/index.html`
